애드가드의 다양한 기능들

효율적인 광고방지를 위해 다양한 기능을 제공합니다.

  • 키워드 감지 가리기/내보내기

    가장 기본적인 유형으로 광고와 관련한 키워드가 감지될 시 즉시 해당 채팅을 가리고 내보냅니다.

  • 채팅 타입 감지(#검색, 카드 등)

    텍스트가 아닌 다른 다른 타입의 광고(#검색, 카드광고, 일정공유, 오픈프로필 공유 등)차단을 위해

    텍스트, 이모티콘, 이미지 등 일상적인 유형을 제외한 다른 유형의 채팅은 즉시 감지하여 가리기 처리합니다.

  • 화이트리스트 설정 기능

    특정 키워드가 포함된 채팅을 입력해도 가리기나 내보내기가 되지 않아야 할 슈퍼유저들을 위해 만들어진 기능입니다.

    "!등록 @유저멘션" 명령어를 통해 방장/ 부방장 권한을 가진 유저가 수동으로 화이트리스트에 추가/삭제/초기화 시킬 수 있습니다.

  • 멘션광고 차단 기능

    멘션을 통해 전체 방 유저들을 호출하는 광고에 대응하기 위해,

    방장, 부방장이 아닌 타 유저가 일정 수 이상의 유저를 멘션할 경우 바로 해당 채팅을 가리고 내보내기 처리합니다.

  • 채팅창 대량 가리기

    카카오톡 자체의 버그를 이용한 광고 도배에 대응하기 위해,

    특정 명령어를 사용하여, 순차적으로 최대 80개의 채팅이 가려질 수 있도록 하는 기능입니다.

  • 입장닉네임 키워드 감지

    "방장", "알리미", "관리자", "매니저" 등등...

    관리자의 닉네임을 사칭하여 입장 후 타사 홍보를 하여 고객을 빼내가는 스파이를 사전차단하기 위해, 특정 키워드가 포함된 닉네임으로 입장 시 키워드를 감지, 들어오는 즉시 바로 내보내는 시스템을 구축했습니다.

  • 내보내기 ON/OFF 기능

    "!모드변경" 명령어를 통해,
    가리기만 진행할지,
    가리기와 내보내기 기능을 모두 진행할지 고객이 직접 선택할 수 있습니다.

  • 로그 파일 제공(.txt 형식)

    매일 가려지는 스팸 내역들은 로그파일에 저장되어, 추후 키워드 설정 등에 유용하게 사용하실 수 있습니다.

  • 키워드 수동 추가 기능

    항상 새로운 키워드로 광고를 시도하는 스파이를 좀 더 효율적으로 막기 위해서 방장, 부방장 권한의 유저가 명령어를 통해 키워드를 직접 추가할 수 있도록 하였습니다.

  • 24시간 365일 작동

    컴퓨터 응용프로그램으로써 실행시키는 것이 아닌 서버에서 작동시키는 방식이라 멈추지 않고 작동합니다.

    주말에는 안심하고 휴식하실 수 있도록 애드가드는 쉬지 않고 돌아갑니다.
